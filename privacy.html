Privacy Policy

Last updated: April 2026

1. Overview

MyCutsApp ("the App") is a personal desktop application that processes local video files and publishes clips to TikTok. This policy explains what data is collected and how it is used.

2. Data We Collect

The App collects and stores the following data locally on your device only:

  • TikTok access token and refresh token — obtained during OAuth authentication, stored in a local configuration file to allow video publishing.
  • TikTok open_id — a unique identifier returned by TikTok after authentication, used to identify the connected account.
  • Video files and metadata — processed locally and never uploaded to any server other than TikTok.

3. Data We Do Not Collect

  • We do not collect names, email addresses, passwords, or any personal information.
  • We do not use analytics, tracking, or advertising of any kind.
  • We do not share any data with third parties other than TikTok (as necessary for publishing).

4. TikTok API

The App integrates with TikTok's Login Kit and Content Posting API. When you authenticate, TikTok may collect data according to its own Privacy Policy. We encourage you to review it.

5. Data Storage & Security

All tokens and configuration are stored in a local JSON file on the user's computer. No data is transmitted to any external server other than TikTok's official API endpoints.

6. Data Deletion

You can delete all stored data at any time by removing the tiktok_config.json file from the App's directory. This will disconnect your TikTok account from the App.
